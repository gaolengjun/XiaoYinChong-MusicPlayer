# XiaoYinChong-MusicPlayer
使用Pyside6开发的音乐播放器

[![pF0RkIe.md.jpg](https://s11.ax1x.com/2024/03/02/pF0RkIe.md.jpg)](https://imgse.com/i/pF0RkIe)

该程序使用Python&Pyside6开发，v1.0版本进度条BUG:进度条与歌曲时间不同步，切歌后不会更新
### 目前功能：
- 播放与暂停音乐
- 上一曲、下一曲和双击列表播放
- 音量控制
### 未来预计更新与修复
- 修复进度条
- 加入自动连播功能
- 加入专辑封面显示
- 加入歌曲评论
- 性能优化
